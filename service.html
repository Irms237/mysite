DDOS PROTECTION

Jaringan Game Hosting kami di semua kami dilindungi oleh perlindungan DDoS untuk memastikan bahwa server Anda tetap online, bahkan selama serangan. Kami memastikan bahwa Hosting Anda dilindungi 24/7 terhadap semua jenis serangan dari luar dan kemudian kami tidak peduli berapa lama serangan itu berlangsung dan seberapa besar itu, Kami tetap bersedia untuk membantu mencegah serangan tersebut.

Beberapa serangan yang Kami Amati

01
UDP Floods

UDP Floods adalah jenis serangan penolakan layanan di mana sejumlah besar paket User Datagram Protocol (UDP) dikirim ke server yang ditargetkan dengan tujuan membebani kemampuan perangkat untuk memproses dan merespons. Firewall yang melindungi server target juga dapat habis akibat UDP Floods, yang mengakibatkan penolakan layanan terhadap lalu lintas yang sah.

02
HTTP GET/POST Floods

HTTP GET/POST Floods adalah jenis serangan di mana sejumlah besar permintaan HTTP GET atau POST dikirim ke server target dengan tujuan membebani kemampuan server untuk memproses permintaan tersebut.

03
UDP Fragmentation

UDP Fragmentation adalah jenis serangan di mana paket UDP yang sangat besar dikirim ke server target, yang kemudian dipecah menjadi fragmen yang lebih kecil. Ini dapat membebani kemampuan server untuk memproses fragmen tersebut.

04
Spoofed SYN

Spoofed SYN adalah jenis serangan di mana paket SYN dengan alamat IP yang dipalsukan dikirim ke server target, yang dapat menyebabkan server menghabiskan sumber daya untuk mencoba membangun koneksi yang tidak pernah selesai.
